## AI-Bloom（自己理解・自己肯定感アプリ）🌻

**MVPリリース日**：2025/12/18  
**本リリース日**：2025/01/31 予定 

**アプリURL**：https://ai-bloom.onrender.com 

**GitHub URL**：https://github.com/Pe-chan8/AI-Bloom

### サービス概要
AI-Bloomは、日々の出来事や感情を記録し、  
AIとの対話を通して「自分を責めずに振り返る」体験を提供する  
自己理解・自己肯定感支援アプリです。

**＜MVP＞**
- 日記投稿 × AIバディからのやさしい返信
- ソーシャルタイプ診断による自己理解
- 複数のAIバディから自身に合ったAIバディを選択

**＜本リリース＞**
- 投稿内容の深掘り
- 自己分析（過去の投稿から強み・苦手な環境などをAIが言語化）
