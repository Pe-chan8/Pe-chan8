## サービス概要

AI-bloom は、日々の小さな頑張りを“やさしく抱きしめ”、 
AIの問いかけで自己理解と自信形成を支える **セルフコンパッション × 自己分析ツール**です。

「今日起きられた」「話しかけてみた」「詰まったけど諦めなかった」
―― そんな誰にも言わないけれど、たしかに乗り越えた一歩を短文で残すだけで、 
AIが“いちばんの味方”になり会話をします。

MVPでは日々の日記に対する共感やチアアップ、
本リリースではAIとの会話を通じて考えや行動の深掘り、自己分析を行い、
”より自分を好きに”、”自信を持って人と向き合える自分に”なることを目的としています。

**制作開始日：** 2025/11/07  

**MVPリリース日：** 2025/12/18
 - 日記投稿 × AIバディからのやさしい返信
 - ソーシャルタイプ診断による自己理解
 - 複数のAIバディから自身に合ったAIバディを選択

**本リリース日：** 2025/01/31 予定
 - 投稿内容の深掘り
 - 自己分析（過去の投稿から強み・苦手な環境などをAIが言語化）

**アプリURL：** https://ai-bloom.onrender.com 

**GitHub URL：** https://github.com/Pe-chan8/AI-Bloom
